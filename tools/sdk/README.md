# Stakied SDK

TypeScript/JavaScript SDK for interacting with Stakied Protocol on Stacks blockchain.

## Installation

```bash
npm install @stakied/sdk
```

## Quick Start

```typescript
import { StakiedSDK } from '@stakied/sdk';

const sdk = new StakiedSDK({ network, contractAddress });
```

See [documentation](../../docs/SDK_GUIDE.md) for full API reference.
